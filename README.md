# 🎮 Mini Games

A collection of fun browser-based mini games built using HTML, CSS, and JavaScript.


Website: https://shriyareddy216.github.io/minigames/
## 🚀 Features

- Simple and responsive UI
- Lightweight and fast
- Play directly in your browser
- Easy to extend with new games

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
minigames/
├── arcadia.html
├── README.md
└── (other assets)
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/shriyareddy216/minigames.git
```

2. Open `arcadia.html` in your browser.
